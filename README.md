# Cola Tech Blog

个人技术博客，基于 Zola 构建。

## 技术栈

- **静态站点生成器**: [Zola](https://www.getzola.org/) - 用 Rust 编写的快速静态站点生成器
- **主题**: [After Dark](https://github.com/getzola/after-dark) - 支持暗色模式的现代主题
- **部署**: GitHub Pages

## 本地开发

```bash
# 启动开发服务器
zola serve

# 构建静态站点
zola build
```
